# Data Warehouse and Analytical Project

This project entails the development of a data warehouse designed to facilitate advanced analytical reporting. It emphasizes the efficient organization and analysis of extensive datasets to derive actionable insights.

## Project Requirements

### Build the Data Warehouse

#### Objective
The objective is to design and implement a scalable data warehouse that integrates data from multiple sources, ensuring data consistency and enabling efficient querying for analytical purposes.

#### Specification
- **Data Sources**: Utilize CSV files containing ERP and CRM tables.
- **Data Quality**: Ensure data accuracy, consistency, and completeness during the ETL process.
- **Integration**: Combine data from both sources into unified, user-friendly data models.
- **Scope**: Historization of tables is not required for this project.
- **Documentation**: Provide clear and detailed documentation of the data model for reference and maintenance.

### BI: Analytics and Reporting

#### Objective
The objective is to create comprehensive business intelligence solutions that leverage the data warehouse to generate insightful analytics and dynamic reporting. This will empower stakeholders to make data-driven decisions effectively.
